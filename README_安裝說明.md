# 物流配送 Netlify 共用系統（更新版）

## 固定地點
- 高雄倉庫：高雄市仁武區成功路152號
- 台南倉庫：台南市東區自由路一段8號
- 台南門市：台南市東區崇善路762號

## 本版功能
1. 商品改為勾選：沙發、餐桌、餐椅、茶几、邊几、掀床架、床架、床底、圓凳、抱枕、床頭櫃。
2. 勾選商品後才顯示數量、撿貨位置、其他地址及商品備註。
3. 每項商品可分別選高雄倉庫、台南倉庫、台南門市或其他地址。
4. 月曆每一天固定顯示第 1～5 組配送；第 6 組起顯示候補警示。
5. 新增物流撿貨表，依撿貨地點分組，並顯示客戶、配送順序、商品及數量。
6. 卸貨及組裝時間由業務手動輸入；系統僅以 Google 車程加上手動作業時間推算下一站抵達時間。
7. 台南線第一站預設 11:00 抵達；硬性與優先時段規則保留。

## 更新既有 Netlify 網站
1. 解壓縮本部署包。
2. 將所有檔案覆蓋到原 GitHub repository。
3. 在 GitHub 按 Commit changes。
4. Netlify 會自動重新部署；網址維持不變。
5. Netlify 環境變數保留：`GOOGLE_MAPS_API_KEY`、選填 `SHARED_PIN`。

## Google 設定
- Google Cloud 必須啟用 Routes API 並開啟帳單。
- API key 放在 Netlify Environment variables 的 `GOOGLE_MAPS_API_KEY`。
- 不要把金鑰直接寫進 index.html。

## 注意
舊版已存資料若沒有 `productItems`，仍會顯示在配送清單，但不會自動出現在撿貨表。新建立的配送資料會完整支援撿貨分組。
