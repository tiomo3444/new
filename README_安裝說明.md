# 物流配送系統 V1.3

本版更新：

- 配送清單新增「編輯」與「刪除」功能。
- 業務可修改日期、時段、客戶、地址、商品、撿貨位置、卸貨及組裝分鐘數與狀態。
- 每日路線不再區分台南線或其他路線。
- 固定由高雄倉庫（高雄市仁武區成功路152號）出發。
- 第一組客戶最早抵達時間為 11:00；較晚的硬性限制優先。
- 下一站時間＝上一站離開時間＋Google 預估車程。
- 月曆日期使用深色底，與第 1～6 組順序清楚區分。
- 卸貨及組裝時間由業務手動輸入並清楚顯示。

## 更新方式

1. 解壓縮本 ZIP。
2. 將資料夾內全部檔案上傳到原 GitHub Repository 並覆蓋同名檔案。
3. Commit changes。
4. 等待 Netlify 自動部署完成。
5. 重新整理原本網址。
