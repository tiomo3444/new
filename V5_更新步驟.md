# V5 免 PIN 修正版

## 本版修正

- 完全移除前端與 Netlify Functions 的 PIN 驗證。
- 改用新的 API 路徑 `/api/v5/...`，避免載入舊版 Functions。
- 加入 no-cache 設定，降低手機瀏覽器載入舊版程式的機率。
- 不需要刪除 Supabase 資料，也不需要重新建立資料庫。

## GitHub 更新

1. 解壓縮 ZIP。
2. GitHub → Add file → Upload files。
3. 上傳解壓縮後的所有檔案並覆蓋。
4. Commit：`V5 完全移除 PIN 驗證`。
5. 等 Netlify 顯示 Published。

## Netlify

`SHARED_PIN` 環境變數可以刪除，也可以暫時保留；V5 完全不會讀取它。

## 手機測試

部署完成後，請先關閉原本的網站分頁，再重新開啟網址。若仍看到舊畫面，使用無痕模式測試一次。
