# 裏物傢俱配送管理平台 V4

本版修改：

- 平台名稱改為「裏物傢俱配送管理平台」。
- 取消共用 PIN，手機與電腦開啟網址即可使用。
- 新增「今日配送」首頁。
- 月曆顯示「業務姓名＋客戶姓名」，例如 `Ti 王小明`。
- 月曆自動從完整地址擷取行政區，例如東區、安南區、永康區。
- 月曆簡易顯示主要商品；多品項顯示「沙發＋2」。
- 待確認、已確認、配送中、已完成使用不同底色。
- 手機版維持單日直式月曆。
- 保留商品勾選、快速狀態更新、編輯、刪除、撿貨表與 Supabase 共用資料。

## 更新 GitHub

將解壓縮後全部檔案上傳到原 Repository 覆蓋，Commit 後等待 Netlify 自動部署。

Netlify 的 SHARED_PIN 環境變數可以保留，也可以刪除；V4 不再使用它。
